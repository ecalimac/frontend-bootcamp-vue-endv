# cuddly-octo-pizza

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

## run DB server

```
in order to use Cuddly's pizza spectacular DB, (in a new terminal) run:

json-server --watch db.json
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
